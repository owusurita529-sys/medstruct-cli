Download Clinstruct-cli

      

Place binaries in docs/downloads and the links below will work automatically.

      
        
          

Linux aarch64 (required)

          clinote-aarch64-unknown-linux-musl.tar.gz         
        
          

Linux x86_64 (optional)

          clinote-x86_64-unknown-linux-musl.tar.gz         
      
      
        

Install

        
# Linux aarch64
curl -LO ./downloads/clinote-aarch64-unknown-linux-musl.tar.gz
tar -xzf clinote-aarch64-unknown-linux-musl.tar.gz
./clinote --help

# Linux x86_64
curl -LO ./downloads/clinote-x86_64-unknown-linux-musl.tar.gz
tar -xzf clinote-x86_64-unknown-linux-musl.tar.gz
./clinote --help
        

You can rename the binary to clinote for convenience.
